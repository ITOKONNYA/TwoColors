## 配色の生成方法

1. PSSCの各色（12のトーン × 12の色相 = 144色）からメインの色をランダムに選ぶ。
2. メインの色に応じて、PCCSの各色から対照的な色を集めた配列を作る。
3. 作った配列からアクセントの色をランダムに選ぶ。

#### PCCS (Practical Color Coordinate System)

日本色彩研究所が開発した表色系。トーンと色相という二つの属性で、色を二次元で表記できることが特徴。
